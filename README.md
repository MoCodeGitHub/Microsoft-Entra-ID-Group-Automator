# Microsoft Entra ID Group Automator (PowerShell)

## 📌 Project Overview

This project demonstrates automated creation of Microsoft Entra ID (Azure AD) security groups using PowerShell.

The script reads group information from a CSV file and automatically provisions security groups in Microsoft Entra ID. This simulates real-world IT administration tasks such as onboarding departments, managing permissions, and automating identity infrastructure.

---

## ⚙️ How It Works

1. A CSV file defines group names and descriptions.
2. PowerShell connects to Microsoft Graph.
3. The script reads each row from the CSV.
4. Security groups are created automatically.
5. Output confirms success or failure.
6. Results can be verified in Microsoft Entra ID.
